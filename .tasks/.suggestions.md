# Upgrade Suggestions

- Add a CI lint step that validates all structural test regexes against their target files on every PR, so regex-drift (destructuring gains a field, import order changes) surfaces as a CI failure rather than a silent mismatch discovered only after merge.
- Share a single `RealtimeClient` instance across `supabase-chat.ts` and `supabase-marketplace.ts` (and any future modules) by extracting `getRealtimeClient` into a shared `lib/supabase-realtime.ts` module; today each module spins up its own WebSocket connection to the same endpoint, doubling socket overhead.
- Export `getDefaultCurrencyForLanguage` from `currency-input.tsx` into `lib/locale-helpers.ts` alongside future locale utilities so the mapping stays reusable across forms (e.g. the future sell/trade form on collection detail screens) without coupling it to the UI component.
